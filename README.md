# Logistics SQL Analysis

## Project Overview

This project analyzes logistics trade and shipment data using SQL to uncover trends in exports, imports, trade balances, and delivery performance.

The goal is to demonstrate practical SQL skills commonly used by data analysts when working with operational and logistics datasets.

## Dataset

The dataset contains shipment records with the following fields:

* Month
* Year
* Port Name
* Port Type
* Product Category
* Export Value
* Import Value
* Trade Balance
* Delivery Days

## SQL Skills Demonstrated

* SELECT
* WHERE
* GROUP BY
* HAVING
* ORDER BY
* LIMIT
* SUM()
* AVG()
* COUNT()

## Business Questions Answered

1. Which month had the lowest total export value?
2. Which ports have average delivery times below 4 days?
3. What is the total trade balance for each product category?
4. How many shipments took more than 5 days?
5. What are the total exports and average delivery times by port type?
6. Which port generated the highest export value?
7. Which product category had the highest average export value?
8. What are the monthly export and import totals?
9. Which port has the fastest average delivery time?
10. Which ports have negative trade balances?

## Key Insights

* Jubail Port generated the highest export value.
* Chemicals recorded the strongest export performance.
* Export activity increased throughout the year.
* Delivery efficiency varied across port types.
* Some ports maintained negative trade balances.
